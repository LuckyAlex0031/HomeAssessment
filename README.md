# HomeAssessment

## Description

This is a simple application that allows you to manage and retrieve company employee information.

## Technologies

- Java
- Spring Boot
- Maven
- JUnit
- Thymeleaf
- Apache Tomcat

## How to run

1. Clone the repository
2. Run `mvn clean install`
3. Run `mvn spring-boot:run`

## Endpoints

- GET /employees
- GET /employees/{id}
- GET /employees/date-range
- POST /employees

## Example

```bash
GET /employees
```


```bash
GET /employees/{id}
```


```bash
GET /employees/date-range
```


```bash
POST /employees
```
